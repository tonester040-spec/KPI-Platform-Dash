# Voice Profile

This directory holds Karissa's writing style profile used to generate email drafts that sound like her.

## Files

- `karissa_voice_profile.json` — Auto-generated by `build_profile.py`. **Do not edit by hand** unless you know what you're doing.
- `samples/` — Drop 30-40 of Karissa's sent emails here as `.txt` files before running `build_profile.py`

## Setup (one-time)

1. Export 30-40 sent emails from Karissa's Gmail
   (variety of recipients, tones, and lengths — see spec for details)

2. Save each email as a plain `.txt` file in `voice/samples/`
   Name them anything: `01.txt`, `02.txt`, etc.

3. Run the profile builder:
   ```
   python email/build_profile.py
   ```

4. Review the printed summary — does it match how she actually writes?

5. The profile is now ready. The draft generator will use it automatically.

## What's in the profile

- Greeting patterns
- Sign-off patterns
- Sentence length preference
- Formality level
- Punctuation style
- Vocabulary patterns
- Paragraph structure
- Distinctive phrases

## Notes

- The `samples/` directory is in `.gitignore` — Karissa's private emails never get committed to the repo
- The profile JSON itself is safe to commit (no actual email content, just style analysis)
- You can manually edit `karissa_voice_profile.json` to fine-tune anything that sounds off
